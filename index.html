Installation

1- Clonez le projet :

C:\...> git clone https://github.com/MohammedAmine/ctdm

2- Placez vous au niveau du répertoire "/citadium" :

C:\...\ctdm> cd citadium

3- Installez les dépendances :

> npm install && npm install -g cordova ionic

4- Testez sur un naviguateur :

> ionic serve

Développement

  • Build:
> gulp

ou

> gulp build
  • Build avec minification & strip debugs :
> gulp build --release
  • Watch et rebuild au changement du scss :
> gulp watch

Gestion des plate-formes mobiles => Emulation => Build => Débogage sur périphériques mobiles => Packaging

  • Ajouter une plate-forme :
> ionic platform add [ios|android]
  • Build :
> ionic build [ios|android]
  • Emulation :
> ionic emulate [ios|android]
  • Débogage sur périphérique mobile:
> ionic run [ios|android]
  • Packaging :
> ionic package [mode] [ios|android]
