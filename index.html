Quick Start

1- Clone project :

> git clone https://github.com/MohammedAmine/pfe

2- Navigate to /citadium folder :

> cd citadium

3- Setup :

> npm install

4- Test in a browser :

> ionic serve

Development

  • Build:
> gulp

or

> gulp build
  • Build with minification & strip debugs :
> gulp build --release
  • Watch and rebuild on change :
> gulp watch

Platform managment, Emulation, Building, Running on real device and Packaging

  • Add platform :
> ionic platform add [ios|android]
  • Build :
> ionic build [ios|android]
  • Emulate :
> ionic emulate [ios|android]
  • Run :
> ionic run [ios|android]
  • Package :
> ionic package [mode] [ios|android]
